# ReactTest (and more package test)

### Get STARTED!!
-------------
```
cd /ReactTest
yarn init
```
### Did you start now?
-------------
`yarn run start`

and connect `localhost:3000`

### Did you start develop?
-------------
`yarn run storybook`

and connect `localhost:3001`
